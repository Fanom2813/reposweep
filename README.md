# RepoSweep

A lightweight desktop app that helps developers reclaim disk space by cleaning up build artifacts, caches, and dependencies across their workspaces.

## What it does

Point RepoSweep at a folder that contains your projects. It scans for known project types, identifies cleanup targets like `node_modules`, `target/`, `.venv`, `build/`, and shows you exactly how much space you can reclaim — then lets you clean with one click.

### Supported project types

Node, Flutter, Rust, Python, Go, .NET, Java, Ruby, PHP, Swift, Elixir, and Git repositories.
